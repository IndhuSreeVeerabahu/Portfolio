# Assets Folder

This folder contains the following files:

## Required Files:

1. **profile.jpg** - Professional profile photo (recommended size: 800x800px, square format)
   - Place your professional headshot here
   - Ensure the image is optimized for web (JPEG format, compressed)

2. **resume.pdf** - Your resume in PDF format
   - Place your downloadable resume here
   - Ensure the file is named exactly `resume.pdf`

## Notes:

- Both files are referenced in the HTML
- Make sure the file names match exactly: `profile.jpg` and `resume.pdf`
- For best results, use a square profile photo (1:1 aspect ratio)
- Optimize images for web to ensure fast loading times
